外边距折叠现象 合并现象

场景：垂直布局 的 块级元素，上下的margin会合并

结果：最终两者距离为margin的最大值

解决方法：避免就好

  • 只给其中一个盒子设置margin

实战

两个盒子 设置外边距 只要设置一个就可以了

外边距折叠现象-塌陷现象

场景：互相嵌套 的 块级元素，紫元素的margin-top 会作用在父元素上

结果：导致父元素一起往下移动

解决方案：

  • 给父元素设置borader-top 或者 padding-top(分隔父子元素的margin-top)
  • 给父元素设置overflow:hidden 推荐这种
  • 转换成行内块元素
  • 设置浮动

行内元素 内外边距 margin padding

如果想通过margin 或 padding 改变行内标签的垂直位置， 无法生效

解决方案：使用行高属性 line-height
